# 后台数据目录

> ⚠️ **此目录下的 Excel 文件包含敏感业务数据，已加入 `.gitignore`，不会上传到 GitHub。**

## 需要的源数据文件

运行本项目需要在此目录下放置以下两个 Excel 文件：

| 文件名 | 说明 | 对应 Sheet |
|---|---|---|
| `场景能力数据列表.xlsx` | 场景能力数据（406 条） | 创新产品发布审核列表 |
| `场景机会数据列表.xlsx` | 场景机会数据（1186 条） | 应用场景发布审核 |

## 文件来源

以上两个文件来自内部数据平台导出，包含企业名称、产品名称、合作方向等业务敏感信息，严禁外传。

## 使用方式

将文件放入本目录后，启动 `python app.py` 即可自动加载。
